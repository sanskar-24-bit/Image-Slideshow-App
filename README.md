Python Image Slideshow Project

Overview

This project is a Python-based Image Slideshow application that allows users to display multiple images in a slideshow format with smooth transitions and automatic image switching. The project is simple, interactive, and useful for learning image processing and GUI development in Python.

Features

- Display images in slideshow format
- Automatic image transitions
- User-friendly interface
- Adjustable slideshow speed
- Support for multiple image formats
- Lightweight and easy to use

Technologies Used

- Python
- Tkinter / OpenCV / PIL (depending on your implementation)
- OS Module

Project Structure

Image-Slideshow/
│── images/
│── slideshow.py
│── requirements.txt
│── README.md

Installation

1. Clone the repository:

git clone https://github.com/your-username/image-slideshow-project.git

2. Navigate to the project folder:

cd image-slideshow-project

3. Install required libraries:

pip install -r requirements.txt

How to Run

Run the Python file using:

python slideshow.py
